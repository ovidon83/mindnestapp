# Thouty - Your Thought Companion

Capture, organize, and act on your thoughts with AI-powered insights.

## Features

- **Capture Thoughts**: Voice or text input to capture thoughts instantly
- **AI-Powered Organization**: Automatically identifies sparks, potentials, and insights
- **Share It**: Generate social media posts for LinkedIn, Twitter, and Instagram
- **To-Do Management**: Track and complete actionable thoughts
- **Mind Review**: Daily and weekly thought reviews

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS
- Zustand for state management
- Supabase for backend
- OpenAI for AI processing

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
- `VITE_OPENAI_API_KEY` (optional)

3. Run development server:
```bash
npm run dev
```
